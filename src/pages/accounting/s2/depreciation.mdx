## Depreciation

- recognition of the <span class="highlight red">reduction</span> in the value of assets<br/>
  &ensp;&ensp;&ensp;-> <span class="highlight">effects are not necessarily irraversible</span>
- the precautionary principle ([S1 - Accounting Principles - Principle of prudence](../s1/accounting-principles#principle-of-prudence))


<Callout type="info">**Provision** = risk of default (not paying)</Callout>

&ensp;&ensp;&ensp;- <span class="highlight orange">receivables</span> (debtors, someone who owes you)<br/>

## Depreciation of assets

<Callout type="info">
    Inventory Value < Original Value <br/>
        &ensp;&ensp;&ensp;= <span className="highlight red">probable loss</span> = **provision**
</Callout>

<Callout type="error" emoji="‼️">
  **Capital gain (inventory value > original value) = no entry**
</Callout>

### Financial assets

200 investement securities at **240€** each and worth now **210€** each.

- Original Unitary Price: <Latex>$240€$</Latex>
- Inventory Unitary Price: <Latex>$210€$</Latex>
- Original Value: <Latex>$240€ \times 200 = 48 000€$</Latex>
- Inventory Value <Latex>$210€ \times 200 = 42 000€$</Latex>

**LOSS** = <Latex>$48 000€ - 42 000€ = 6 000€$</Latex>

<Callout type="error" emoji="‼️">
  **LOSS** = <Latex>$(\text\{Original Value} - \text\{Inventory Value}) \times \text\{Quantity\}$</Latex><br/>
  = <Latex>$\text\{Original Value}  \times \text\{Quantity\} - \text\{Inventory Value} \times \text\{Quantity\}$</Latex>
</Callout>

### Inventory stock 

500 iPhones X at **900€** each and worth now **797€** each.

- Original Unitary Price: <Latex>$900€$</Latex>
- Inventory Unitary Price: <Latex>$797€$</Latex>
- Original Value: <Latex>$900€ \times 500 = 450 000€$</Latex>
- Inventory Value <Latex>$797€ \times 500 = 398 500€$</Latex>

**LOSS** = <Latex>$450 000€ - 398 500€ = 51 500€$</Latex>
